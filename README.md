# 🎮 Video Game Sales Analysis (1980–2024)

An end-to-end data analysis project built completely using AI — no manual coding.

---

## 📂 Dataset
Raw sales data of 18,922 video games across consoles, genres, regions and publishers.
Source: [Kaggle](https://www.kaggle.com)

---

## 🤖 Tools Used

| Tool | What I used it for |
|---|---|
| **ChatGPT** | Generated business & trend questions to explore the data |
| **Julius AI** | Uploaded the CSV, asked questions, got charts + full report |
| **Polymer** | Instant interactive dashboards from raw data |
| **Anomaly AI** | Auto data cleaning and visual exploration |
| **Claude** | Summarized findings and wrote the documentation |

---

## 📊 What the Data Showed

- 🏆 GTA V on PS3 — highest selling game at **20.32M copies**
- 🕹️ PS2 is the all-time best console with **1,027M in total sales**
- 🎯 Sports, Action & Shooter = **50%+ of the entire market**
- 🌍 North America drives **50.7%** of all global game sales
- 📈 Market peaked in **2008** and has been declining since

---

## 🚀 How to Do This Yourself

1. Grab any dataset from Kaggle
2. Ask **ChatGPT** to generate business & trend questions for your data
3. Upload to **Julius AI** and ask those questions — get instant charts & reports
4. Drop the same file into **Polymer** for dashboards
5. Use **Claude** to write up your findings
6. Push to GitHub 🎉
